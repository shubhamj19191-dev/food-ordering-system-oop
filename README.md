# 🍔 QuickBite — Food Delivery System (OOP Project)

🚀 A Python-based food delivery system built as part of my learning journey in Object-Oriented Programming (OOP).

This project simulates a real-world application like Swiggy/Zomato and is being developed step-by-step to understand system design and clean coding practices.

---

## ✨ Current Features
- 👤 Customer management (signup, wallet)
- 💰 Wallet system
- 🛒 Order placement
- 💳 Payment system (UPI / Card / COD)
- 📦 Basic delivery flow
- ⭐ Rating system (in progress)

---

## 🧠 What I’m Learning
- Object-Oriented Programming (OOP)
- Inheritance & Method Overriding
- Class Design & System Architecture
- Writing clean and maintainable code
